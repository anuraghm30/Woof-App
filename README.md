# 🐶 Woof App

An Android application built using **Kotlin** and **Jetpack Compose** that displays a list of dogs along with their images, age, and favorite activities.
The project demonstrates **modern Android UI development using Material Design 3 and Jetpack Compose components**.

---

## 📱 Features

* 🐕 Displays a **scrollable list of dogs**
* 🖼️ Shows **dog image, name, and age**
* ⬇️ **Expandable cards** to reveal favorite activities
* 🎨 **Material 3 UI design**
* 🌗 Supports **light and dark theme**
* ⚡ Smooth **Compose animations**

---

## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Material Design 3**
* **Android Studio**

---

## 📚 Concepts Used

* Composable Functions
* LazyColumn for Lists
* Row and Column Layouts
* State Management
* Material Theming
* UI Animations
